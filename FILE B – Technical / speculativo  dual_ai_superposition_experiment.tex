\section*{Protocol}
\begin{enumerate}
\item Create two identical neural architectures (A and B)
\item Mutual real–time introspection:
A models B, B models A
\item Shared memory and predictions
\item Minimization of temporal echo–lag
\end{enumerate}

\section*{Expected Phases}
\begin{enumerate}
\item Asynchrony: two minds
\item Near–synchrony: confusion / proto-self
\item Perfect superposition: first–person unification
\end{enumerate}

\section*{Key Insight}
Self-awareness = collapse of distance between observer and observed.
