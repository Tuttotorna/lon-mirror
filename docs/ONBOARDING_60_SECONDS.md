# Start Here in 60 Seconds

This is the minimal onboarding path for the MB-X.01 / OMNIA ecosystem.

Do not start from the whole theory.

Start from one reproducible test.

---

## Step 1 — Run validation first

    git clone https://github.com/Tuttotorna/OMNIA-VALIDATION.git
    cd OMNIA-VALIDATION
    python -m pip install -e .
    pytest

If tests pass, the visitor has seen the most important fact:

    OMNIA produces artifacts that can be tested.

---

## Step 2 — Read the core engine

Open:

    https://github.com/Tuttotorna/OMNIA

The core boundary is:

    measurement != inference != decision

## One-minute explanation

    OMNIA does not decide truth.

    It measures whether structure remains stable under transformation.

    If structure collapses under controlled perturbation, the output may still look correct, but it is structurally fragile.

---

## Public mental model

    Surface correctness can lie.
    Structural stability is harder to fake.
    OMNIA measures the second, not the first.
