IT · Sintesi

MB-X.01 è un’infrastruttura logico-computazionale per audit riproducibili del ragionamento. Misura coerenza e rischio semantico su testi e decisioni con la catena metrica: TruthΩ → Co⁺ → Score⁺. Supporti: Lya (ledger append-only), Omniabase (osservazione multibase), Third Observer (verifica pubblica), UPE (surprisal cumulativo), Hypercoherence H(X).

Formule chiave

TruthΩ = -√( ε + (1 − C)² + ((B + I)/2)² ),   ε > 0
Co⁺     = exp(TruthΩ) ∈ (0,1]
Score⁺  = (C · Co⁺) − (B + I)/2

H(X)    = tanh( (1/N) · Σ_i [ (Ω_i · C_i) / (1 − D_i) ] )
