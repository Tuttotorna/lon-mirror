# Landing Scope

This repository is the public entrance of the MB-X.01 / OMNIA ecosystem.

Its root README must remain short, navigable and onboarding-first.

The complete theory, experiments, legacy material, notebooks and implementation details should not overload the landing page.

Canonical rule:

    root = entrance
    docs = explanation
    examples = demonstration
    archive = history
    lab = experiments

If a section explains philosophy, foundation, detailed architecture, or historical context, it belongs in docs.

If a section helps a first-time visitor decide where to click next, it belongs in README.
