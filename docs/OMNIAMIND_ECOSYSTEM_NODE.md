## Role

OMNIAMIND organizes cognitive or reasoning pipelines around structural measurement.

It does not measure directly.

It does not decide truth.

It does not claim consciousness.

Its correct position is:

```text
OMNIAMIND = orchestration
OMNIA     = measurement
LIMIT     = interruption boundary
Decision  = external layer
```

## Safe interpretation

OMNIAMIND is not a mind in the sentient sense.

It is a structural interface for organizing cognition-like analytical processes before, during, or after measurement.
