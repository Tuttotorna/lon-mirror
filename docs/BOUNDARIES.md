# Boundaries

## Core boundary

```text
measurement != inference != decision
```

This ecosystem must preserve that separation.

## What measurement can do

Measurement can:

- expose structural instability
- detect fragility
- detect non-admissible output form
- compare behavior under transformation
- detect exhaustion or terminal boundary
- produce signals for external layers

## What measurement cannot do

Measurement cannot automatically:

- decide truth
- decide meaning
- replace semantic evaluation
- replace domain expertise
- select an action
- prove correctness

## Layer boundaries

```text
OMNIABASE   = representation observation
OMNIA       = structural measurement
OMNIAMIND   = orchestration
omnia-limit = terminal boundary certification
Decision    = external
```

## Safe language

Use:

- structural measurement
- post-hoc diagnostics
- invariance under transformation
- fragility detection
- terminal boundary
- external decision layer

Avoid claiming:

- artificial consciousness
- autonomous truth engine
- universal proof system
- automatic semantic correctness
- final decision authority

## Final rule

The ecosystem is strongest when it does not overclaim.

Its value is structural clarity.
